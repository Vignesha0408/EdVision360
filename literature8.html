Literature Review 8

Project Name: Secure and Scalable School Data Platform (MERN)

Summary: This paper presents a MERN-based platform to manage school data ingestion, access control, analytics APIs, and dashboards with strong security and scalability.

Key Outcomes:

  • Secure role-based data access
  • Scalable APIs for ingestion and analytics
  • Protected dashboards for stakeholders

Technical Features:

  • MongoDB, Express.js, React, Node.js
  • JWT auth, encryption, input validation
  • CI/CD and horizontal scaling
