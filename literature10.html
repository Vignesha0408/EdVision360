Literature Review 10

Project Name: End-to-End Architecture for Student Performance Intelligence

Summary: This paper outlines an end-to-end architecture: data import (PAT, SAT, attendance) → preprocessing → predictive models → dashboards → alerts (FCM) with Cordova Android app, emphasizing scalability and security.

Key Outcomes:

  • Modular services for ETL, ML, dashboards, alerts
  • Android Cordova app integration
  • Security and governance considerations

Technical Features:

  • Data schemas, ETL pipelines, ML serving
  • FCM messaging, Cordova plugins
  • RBAC, encryption, audit logging
