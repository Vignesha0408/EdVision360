Literature Review 2

Project Name: Predictive Models for Student Performance

Summary: This research benchmarks ML and statistical models (e.g., Logistic Regression, Random Forest, XGBoost) to predict student outcomes using PAT, SAT, and attendance features with explainability.

Key Outcomes:

  • Model comparison and accuracy metrics (AUC, F1)
  • Feature importance and SHAP-based insights
  • Risk segmentation for dashboard indicators
  • Baseline + production-ready pipelines

Technical Features:

  • Scikit-learn/XGBoost pipelines
  • Cross-validation and hyperparameter tuning
  • Explainability via SHAP/LIME
